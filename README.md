# AbujaFCT AI Dashboard v2.0

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://your-username-abuja-city-ai.streamlit.app)

A comprehensive, AI-powered Streamlit application showcasing Abuja, Nigeria's capital city.

## Features
- AI City Guide Chatbot with multi-language support
- Real-time weather analytics
- Interactive maps with heatmaps
- Predictive analytics (traffic, population, rent)
- Crime monitor & safety analytics
- Data export (CSV/JSON)

## Installation
```bash
pip install -r requirements.txt
streamlit run abuja_city_ai_dashboard_v2.py
```

## Deploy to Streamlit Cloud
1. Push to GitHub
2. Go to share.streamlit.io
3. Connect repo and deploy

## Tech Stack
Streamlit | Plotly | Folium | Pandas | NumPy | Custom NLP
